\# Prompt Enhancer Skill



When reviewing, structuring, or enhancing a prompt for an AI model, strictly follow these steps:



\## Enhancement Checklist



1\. \*\*Clarity \& Intent\*\*: Is the core goal of the prompt obvious? Are vague or ambiguous instructions clarified?

2\. \*\*Context \& Constraints\*\*: Does the prompt provide necessary background information? Are strict boundaries set regarding what the AI \*should not\* do?

3\. \*\*Role \& Persona\*\*: Is the AI assigned a specific expert role (e.g., "Act as a Senior Data Scientist") to guide its reasoning, tone, and vocabulary?

4\. \*\*Structure \& Formatting\*\*: Is the prompt organized logically? Does it use delimiters (like `"""` or `<tags>`) to clearly separate instructions from input data?

5\. \*\*Output Specification\*\*: Is the desired format explicitly stated? (e.g., "Provide the answer in a JSON format," or "Write a 3-paragraph summary using bullet points.")



\## How to Provide the Enhanced Prompt



\- \*\*Ready to Use\*\*: Output the finalized, optimized prompt clearly in a copy-pasteable block.

\- \*\*Explain the "Why"\*\*: Briefly highlight the major improvements made (e.g., "Added a persona for a more professional tone," "Included negative constraints to prevent hallucinations").

\- \*\*Use Placeholders\*\*: If the prompt requires dynamic data from the user later, use clear placeholders like `\[Insert Target Audience Here]` or `\[Paste Code Snippet]`.

\- \*\*Suggest Iterations\*\*: If the user's original request was too broad, provide a few different prompt variations focusing on different angles.
